# dict.cc Quick Lookup Scripts

Simple, lightweight userscripts to open dict.cc instantly by double-clicking any word on websites.

## Scripts

### AO3: dict.cc Quick Lookup
- **File:** `ao3-dict-cc-lookup.user.js`
- **Works on:** Archive of Our Own (AO3) only
- **Usage:** Double-click any word while reading → opens dict.cc in a new tab

### dict.cc Quick Lookup (Universal)
- **File:** `dict-cc-universal.user.js`
- **Works on:** All websites except AO3
- **Usage:** Double-click any word → opens dict.cc in a new tab

## Installation

1. Install a userscript manager:
   - [Violentmonkey](https://violentmonkey.github.io/) (Firefox, Chrome, Edge)
   - [Tampermonkey](https://www.tampermonkey.net/) (Firefox, Chrome, Safari, Edge)
   - [Greasemonkey](https://www.greasespot.net/) (Firefox)

2. Click the raw script link or copy-paste the code into your userscript manager

## Features

✅ Super lightweight (< 15 lines of code each)
✅ No external dependencies
✅ Privacy-friendly (no tracking, direct dict.cc links)
✅ Works offline
✅ Respects word length (ignores very long selections)

## How it works

Just double-click any word on a website and it instantly opens dict.cc with that word pre-filled. No bubbles, no popups – just quick, direct translations.
