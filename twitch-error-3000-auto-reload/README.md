# Twitch Error 3000 Auto-Reload

Automatically reloads the Twitch page when Error 3000 occurs.

## Usage

Simply install the script and watch Twitch. When Error 3000 appears, the page automatically reloads.

## Features

✅ Auto-detects Error 3000
✅ Automatic page reload
✅ Lightweight and fast
✅ No configuration needed

## Installation

1. Install a userscript manager:
   - [Violentmonkey](https://violentmonkey.github.io/)
   - [Tampermonkey](https://www.tampermonkey.net/)
   - [Greasemonkey](https://www.greasespot.net/)

2. Copy-paste the script into your userscript manager
