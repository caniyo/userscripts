# My Userscripts

A collection of simple, lightweight userscripts by caniyo.

## Scripts

- **[dict.cc Quick Lookup](./dict-cc/)** – Double-click any word to instantly open dict.cc translations
  - Works on AO3 and all other websites
  - Privacy-friendly, no tracking

## Installation

Install a userscript manager first:
- [Violentmonkey](https://violentmonkey.github.io/) (recommended for Firefox)
- [Tampermonkey](https://www.tampermonkey.net/)
- [Greasemonkey](https://www.greasespot.net/)

Then follow the installation instructions in each script's folder.

## Philosophy

All scripts here follow these principles:
- **Lightweight** – minimal code, maximum functionality
- **Simple** – no unnecessary complexity
- **Privacy-first** – no tracking, no external APIs
- **Reliable** – works everywhere, no dependencies
